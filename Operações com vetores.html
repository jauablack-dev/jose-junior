🔹 3. Enviar para o GitHub (2 opções)

✅ Opção A: Pelo GitHub Desktop (mais fácil)

  1. Baixe GitHub Desktop.

  2. Clique em File > Add Local Repository e selecione a pasta do seu site.

  3. Clique em Publish repository.

  4. Pronto! Seu site vai aparecer no GitHub.

✅ Opção B: Pelo Git + Terminal

  1. Abra a pasta do projeto no terminal.

  2. Digite os comandos:

bash
git init
git add .
git commit -m "Primeiro commit - site pronto"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/meu-site.git
git push -u origin main

🔹 4. Ativar GitHub Pages (para abrir no navegador)

  1. Vá até seu repositório no GitHub.

  2. Clique em Settings > Pages.

  3. Em Branch, escolha main e salve.

arduino
